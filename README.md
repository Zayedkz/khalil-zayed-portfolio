# Khalil Zayed Portfolio

Static portfolio website for Khalil Zayed, focused on GenAI platforms, enterprise software, developer productivity, and AI product engineering projects.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Content

- Profile summary
- Kinectrics professional experience
- Ontario Tech University education
- Technical skills
- AI portfolio project showcase
- Contact links

## Deployment

The site is static and can be deployed with GitHub Pages, Netlify, Vercel, Azure Static Web Apps, or any static file host.

Expected GitHub Pages URL after Pages is enabled:

```text
https://zayedkz.github.io/khalil-zayed-portfolio/
```
