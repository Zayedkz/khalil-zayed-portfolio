AI Code Review Agent

GitHub webhook service for automated pull request triage, with signed delivery verification, normalized review events, idempotent storage, and deterministic risk findings.

View GitHub Portfolio

Overview

A review agent starts with trustworthy event intake.

Problem

Pull requests accumulate repetitive review concerns: large diffs, missing tests, debug markers, placeholder work, and risky secret-handling patterns.

Implemented

  • Express webhook endpoint with raw-body HMAC verification
  • Zod validation and normalized pull request event model
  • Idempotent delivery handling by GitHub delivery ID
  • Deterministic findings with severity and recommendations
