Architecture Review

AI Architecture Review Assistant

AI-ready architecture review service for Mermaid system designs. It extracts requirements, flags architecture findings and SPOFs, recommends scaling paths, and reports tradeoffs from a submitted design.

View GitHub Portfolio

Overview

System design review made reproducible.

Problem

Architecture reviews often depend on informal checklists and reviewer memory. Important questions about scaling, availability, security, and tradeoffs can be missed when designs are reviewed manually.

Implemented

  • Mermaid node and edge extraction without external parsing services
  • Requirements extraction from diagram labels and context
  • Deterministic findings across scalability, availability, reliability, and security
  • SPOF risks, scaling recommendations, tradeoffs, and overall risk level
